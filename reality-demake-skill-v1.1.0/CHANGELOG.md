# Changelog

## [1.1.0] - 2026-07-23

### Added

- Chinese and English skill versions
- Explicit trigger conditions and default behavior
- Four scene adapters: urban, nature, environmental portrait, and props/food
- Light, Standard, and Heavy intensity presets
- Anchor-preservation rules for people, architecture, objects, and text
- Automatic repair loop for common failure cases
- Pre-delivery quality checklist
- Copy-ready full and compact prompts
- Design-principles and troubleshooting documentation
- GitHub-ready repository structure
- Copyright, privacy, and trademark guidance

### Changed

- Refined the core definition from a single style prompt into a repeatable image-remake workflow
- Made Standard intensity the default
- Clarified that the effect must come from rebuilding geometry and materials rather than pixelation or filtering

## [1.0.0] - 2026-07

- Initial public RED Skill release
- Core low-poly / low-resolution / early Source-engine prompt
- Basic intensity and repair instructions
