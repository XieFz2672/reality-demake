# 设计原则

## 1. 结构降级优先于画面降质

真正的“降模”来自几何结构、贴图和光影的重新构建。单纯降低清晰度、增加像素、噪点或压缩痕迹，只能作为辅助。

## 2. 识别度优先于粗糙度

画面可以粗糙，但主体不能消失，人物不能变成陌生人，场景不能被重新布局。

## 3. 低模不等于卡通

避免圆润、可爱、玩具感和现代卡通渲染。低模应该保留旧电脑 3D 游戏的硬边、有限贴图和不自然光影。

## 4. 可探索空间感

街道、入口、台阶、走廊、溪流和建筑之间应保留可通行、可探索的空间关系，让结果像真实关卡，而不是平面海报。

## 5. 时代感重于品牌复刻

可以参考 2004 年左右的引擎特征，但不复制具体角色、地图、Logo 或专有资产。
