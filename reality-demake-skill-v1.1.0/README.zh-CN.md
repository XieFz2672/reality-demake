# 现实太高清了，给它降个模

> 把现实照片重新建模成 2004 年老游戏截图，不是滤镜，是降模。

这是一个免费公开的图片处理 Skill。它会先理解照片里的主体、构图、视角和空间关系，再把人物、建筑、树木、道路、家具、食物与物体重新诠释为早期 3D 游戏资产：低多边形模型、低分辨率贴图、生硬光影，以及略显笨拙但可以探索的旧游戏地图感。

## v1.1.0 完善内容

相比最初以一段 Prompt 为核心的版本，这次增加了：

1. **明确的执行流程**：先分析照片，再选择强度和场景适配方式。
2. **主体保护规则**：锁定人物身份、姿势、构图、建筑轮廓与关键道具。
3. **三档强度**：轻度、标准、重度；默认使用标准档。
4. **四类场景适配器**：城市建筑、自然环境、人物环境照、物品与食物。
5. **自动修复回路**：处理“不够低模、太像滤镜、人物变脸、主体丢失”等问题。
6. **质量检查表**：判断结果是否真正完成了“重新建模”。
7. **中英文双版本**：方便上传 RED Skill、发布 GitHub，以及在不同 AI 助手中使用。
8. **开源仓库结构**：包含 Prompt、故障排查、示例规范、贡献指南和更新记录。

## 适合的照片

优先选择有清晰空间关系的现实场景：

- 街景、小区、公园、校园
- 商场、便利店、地铁站、停车场、楼梯间
- 城市广场、海边、山路、庭院
- 人物位于完整环境中的旅行照
- 可以被理解为游戏道具的食物或物品

## 不太适合的照片

- 只有脸部的大头自拍
- 背景几乎为空的照片
- 主体极小、空间关系不清楚的照片
- 已经过度精修、细节失真的人像
- 需要精确还原小字、商标或复杂招牌的画面

## 快速使用

1. 上传一张照片。
2. 告诉 AI：“使用《现实太高清了，给它降个模》处理这张图。”
3. 没有额外要求时，Skill 默认：
   - 保留原始比例；
   - 使用标准降模强度；
   - 严格保留主体、构图、视角和空间布局；
   - 不新增文字、人物或无关物体。
4. 结果不理想时，根据问题调用对应的修复补丁。
