# Examples Directory

Add before/after images that you have the right to publish. Examples help users understand the intended result.

## Suggested structure

```text
examples/
├── urban/
│   ├── 01-before.jpg
│   └── 01-after.jpg
├── nature/
├── portrait/
└── props-food/
```

## Recommended metadata for each example

- Scene type
- Intensity preset
- Extra repair patches
- Generation platform or model (optional)
- Known issues

## Privacy reminder

Upload only images you own or are authorized to publish. Obtain consent or remove sensitive information when photos contain other people’s faces, addresses, license plates, order details, or other private data.
