# Contributing / 贡献指南

感谢你帮助改进 Reality Demake。

## 可以贡献什么

- 新的场景适配器
- 更稳定的修复补丁词
- 有授权的前后对比示例
- 中英文文案修正
- 不同图片生成模型的兼容性反馈
- 失败案例与复现步骤

## 提交原则

1. 不提交无权公开的图片。
2. 不上传真实地址、车牌、订单信息等敏感内容。
3. 不提交受版权保护的游戏资产、角色、Logo 或完整地图。
4. 示例尽量说明使用的强度、补丁和生成平台。
5. 修改核心风格定义时，请说明为什么它仍属于“现实降模”，而不是另一种视觉风格。

## Pull Request 建议

- 一个 PR 尽量只解决一个问题。
- 更新文档时，尽量同步修改中英文版本。
- 用户可见行为变化应更新 `CHANGELOG.md`。
