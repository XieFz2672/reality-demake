# Reality Demake / 现实太高清了，给它降个模

A free and open image-editing skill for remaking real-world photos as 2004-era low-poly game screenshots.

一个免费公开的图片处理 Skill：把现实照片重新建模成 2004 年老游戏截图，不是滤镜，是降模。
